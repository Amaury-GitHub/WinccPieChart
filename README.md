# WinccPieChart
.NET开发的Wincc自定义饼图控件

可以定义图表的长宽尺寸并自动缩放,便于wincc调用时方便布局<br>
可以定义图表的背景色,便于wincc调用时融入背景,VS中可以使用透明背景,透明背景这个参数导入wincc后会失效<br>
可以定义标签显示的字体,便于融入wincc的整体风格<br>
可以定义使用2D饼图或者3D饼图<br>
可以定义是否显示元素的名称<br>
可以定义是否显示元素的数值<br>
可以定义元素的标签显示在饼图内或者外面<br>
可以扩展饼图的元素,每个元素的标签与颜色都可以自定义,最多8个,应该够用了<br>
